# 塩津 裕康：日本唯一のCO-OPインストラクター、「子どもの参加」を追求する作業療法士・研究者

## 概要

**塩津 裕康（しおづ ひろやす / Hiroyasu Shiozu）** は、**名古屋市立大学 医学部保健医療学科 リハビリテーション学専攻 講師**。X（旧Twitter）では **@H_SHIOZU** として発信。

1985年、三重県生まれ。2008年に作業療法士免許取得後、川崎医療福祉大学大学院で**博士（リハビリテーション学）**を取得（2015年）。**2020年、International Cognitive Approaches Network（ICAN）から認定を受け、日本人初の認定CO-OPセラピストとなり、後にインストラクター資格も取得**。国内外で唯一無二の立ち位置から、CO-OPアプローチの臨床応用・研究・普及教育を主導する。

研究室は **Go2E（Guidance for Occupational Enablement and Engagement）** と名付けられ、「生態学的視点から人間の作業（Occupation）への理解を深め、その可能化と関与の指針を発見すること」をミッションに掲げる。2025年10月より、**JST RISTEX（科学技術振興機構 社会技術研究開発センター）** の大型プロジェクト研究代表者として、飛騨市長と協働し全国初のインクルーシブ教育システム確立に挑んでいる。

## 研究分野・専門テーマ

### 研究室の軸：「生態学的視点から作業の可能化を探る」

塩津氏の研究の基盤には「子どもの作業（Occupation）への参加を生態学的視点から理解し、可能化するための指針を見つけること」というビジョンがある。研究キーワード（12件）は CO-OP approach、作業中心の実践（OCP）、Pragmatism、Enactivism、Relationism、ダイナミックシステム理論、作業発達（Occupational Development）、生態学的視点、作業科学、作業療法、遊び、特別支援教育。

### 1. CO-OPアプローチの臨床応用・研究

**CO-OP（Cognitive Orientation to daily Occupational Performance）アプローチ**とは、カナダのポラタイコ博士らによって開発された、子ども自身が目標を選び、認知ストラテジーを用いて課題解決しスキルを習得する、子ども中心の問題解決型介入法。発達性協調運動症（DCD）・自閉スペクトラム症（ASD）・脳性麻痺・注意欠如多動症（ADHD）・脳卒中・呼吸器疾患など幅広い対象に応用可能で、塩津氏はその**国内唯一のインストラクター**として、小児リハビリ病院から学校・遠隔医療（Telehealth）まで多様な現場で実践・研究を展開している。

主な研究テーマ：
- 小児リハビリ病院・児童発達支援・放課後等デイサービスにおけるCO-OPの実践研究
- 学校作業療法（特別支援教育との連携）へのCO-OP応用
- CO-OPを基盤とした遠隔作業療法（Telehealth）の実践研究
- 統合失調症・脳卒中・呼吸器疾患等の成人領域へのCO-OP拡張
- 少年院におけるCO-OPを基盤とした作業療法実践

### 2. 学校作業療法：飛騨市モデル（HIDAモデル）の確立

岐阜県飛騨市との協働により、全国初の試みとして**すべての小中学校に作業療法室を設置**する「飛騨市モデル（HIDAモデル）」を主導・監修。過疎化・少子高齢化が進む小さな自治体において、教育・保健医療・行政・福祉の四者が連携して「誰も取りこぼさない」インクルーシブ教育を実現する先駆的モデルとして全国的に注目を集めている。

現在は**JST RISTEX「SDGsの達成に向けた共創的研究開発プログラム（シナリオ創出フェーズ）」**（2025年10月〜2027年9月）の研究代表者として、協働実施者の都竹淳也飛騨市長とともに「**教育—保健医療—行政連携によるインクルーシブ教育システムの確立と多地域展開**」に取り組んでいる。プロジェクトHP：https://www.xinclusive.net/

### 3. 作業発達（Occupational Development）研究

子どもが日々の作業（遊び・学習・身辺自立など）にどう発達的に関わり、その参加がどのように形成されるかを探る「作業発達」研究。Pragmatism・Enactivism・ダイナミックシステム理論といった哲学・理論的基盤と、OPC（Occupational Performance Coaching）を用いた子育て支援実践を組み合わせ、子ども—環境—作業の三者相互作用を記述・支援する独自の枠組みを構築している。

### 4. OPC（Occupational Performance Coaching）を用いた子育て支援

神経発達症（発達障害）を持つ子どもの親に対し、OPCを活用した参加支援戦略を研究。2024年には子どもの参加戦略に関する探索的研究を国際誌 *Children* に発表（筆頭著者・責任著者）。

### 5. 評価ツール・作業遂行分析の研究

スクールAMPS（学校版作業パフォーマンス・動機付け評価）・AMPS・ESI・COPMなどの標準化評価ツールの学校・臨床現場での適用研究や、**般化・転移アセスメントツールの開発**にも取り組む。

## まとめ

塩津裕康氏は、「子どもの参加を可能にする」というテーマに徹底的に取り組む、日本の小児作業療法・学校作業療法のトップランナーである。

**日本人で唯一のCO-OPインストラクター**という立場から、個別の臨床実践にとどまらず、学校・地域・行政を巻き込んだシステム変革に向けた研究と実践を一体的に展開。飛騨市との協働による「HIDAモデル」確立、JST RISTEX大型プロジェクト代表、日本特殊教育学会実践研究賞受賞など、臨床・研究・政策提言の三位一体で日本のインクルーシブ教育推進に貢献している。

**論文48編・書籍7冊（著者・監訳・監修）・学会発表94件以上・受賞4件・委員歴10件**という業績は、作業療法士・研究者・教育者として際立った存在感を示している。
